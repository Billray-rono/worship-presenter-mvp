# Worship Presenter MVP (Phase 1 Starter)

## Requirements
- JDK 21
- Gradle 8+

## Run
```powershell
gradle run
```

## Build installer (Windows)
```powershell
gradle jpackage
```

## Current scope
- JavaFX operator window
- Program output window (second screen aware)
- Basic domain/use-case/repository structure
- SQLite connection + schema bootstrap
- Kotlin + Java interoperability example
